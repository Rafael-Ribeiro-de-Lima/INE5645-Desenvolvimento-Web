    Dicionário de Elementos HTML

    Tag Descrição
    <!DOCTYPE> Define o tipo de documento HTML.
    <html> Elemento raiz que contém todo o conteúdo da página HTML.
    <head> Contém metadados (informações sobre o documento), como o título e links para arquivos CSS.
    <meta> Define metadados como o charset e a compatibilidade do documento.
    <link> Define a relação entre o documento atual e um recurso externo (como um arquivo CSS).
    <title> Especifica o título do documento, que aparece na aba do navegador.
    <body> Contém todo o conteúdo visível da página web, como textos, imagens, e seções.
    <header> Define o cabeçalho da página, geralmente contendo navegação ou título principal.
    <nav> Define uma seção de navegação que normalmente contém links de navegação.
    <menu> Um container para um conjunto de itens de menu, como uma barra de navegação.
    <li> Representa um item em uma lista.
    <a> Define um hyperlink que permite a navegação entre páginas ou seções.
    <aside> Define um conteúdo auxiliar ou complementar, como informações adicionais ou biografia.
    <h1> Define o nível mais alto de título em HTML, geralmente usado para títulos principais de seções.
    <div> Um container genérico para agrupar conteúdo.
    <img> Incorpora uma imagem na página.
    <p> Define um parágrafo de texto.
    <section> Define uma seção de conteúdo, como um capítulo ou parte da página.
    <ul> Define uma lista não ordenada de itens.
    <b> Aplica negrito ao texto.
    <footer> Define o rodapé da página, geralmente contendo informações como direitos autorais ou links.
